Вызов staff.main() вернет словарь сотрудников в значених количество публикаций
COUNT_PAGE задает количество страниц для парсинга публикаций. Всего их там 35 я настроил 5 (для эконимии времени)
- Для vscode
### options.add_argument("parser/user")
- Для pyCharm
### options.add_argument("user")
VsCode и pyCharm по разному определяют текущую директорию. Поэтому для них разные настройки

В функции get_next_page стоит двухминутная пауза. Это для авторизации. После авторизации в "user" будет сохранена сессия и паузу можно уменьшить до 2 секунд
